# EchoSnake  By Echonessy

> this is my snake game

###用法
``` bash
Html:
<div class="EleBox" id="Element"></div>
<p class="Start" id="Start">开始</p>

Js:
 var Snake =  new EchoSnake();
    Snake.Config({
        Ele:'#Element', // 绑定元素主体
        Start:'#Start', //启动按钮
        Matrix: 20 // 矩阵大小为20*20 
    });
    
```
[Demo演示地址]( https://echonessy.github.io/EchoSnake/)
